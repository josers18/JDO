How this home page thinks

Two colors carry the whole model — teal = you act (CRM writes), violet = AI acts (summarize / recommend).

When we use agentic summarization

To compress the book into something a banker can read in seconds — never to invent, always cite sources & confidence.

  • Daily brief hero — synthesizes 24 data sources into one paragraph + confidence
  • "Prep me" sheet — relationship recap, talk track, key facts
  • "Why?" reasoning — the evidence behind a ranking
  • Portfolio pulse narrative & email draft generation

When we use recommendations

To prescribe a specific next move the banker can execute or dismiss — always with a reason and a one-click action.

  • Priority queue — ranked "who to act on & why"
  • Recommended actions — pre-drafted task / email / call / case
  • Life events → suggested plays
  • Per-row next-best-action chips

CRM actionability → real writes
