## §2 `briefs` subcommand implementation

```python
def _run_briefs(args: argparse.Namespace) -> int:
    """Generate per-banker brief MD files from live org data."""
    if args.target_org is None:
        print("--target-org is required", file=sys.stderr)
        return 2

    runner = SfRunner(args.target_org)
    rm_pool = yaml.safe_load(Path(args.config_dir) / "rm_pool.yaml")

    # Optionally filter to a single banker via --rm
    rms = rm_pool["rms"]
    if args.rm:
        rms = {k: v for k, v in rms.items() if args.rm in (v["name"], v["user_id"], k)}
        if not rms:
            print(f"No banker matches --rm {args.rm!r}", file=sys.stderr)
            return 2

    output_dir = Path(args.output)
    output_dir.mkdir(parents=True, exist_ok=True)

    from customer_hydration.briefs import generate_brief, generate_index

    summary_rows = []
    for slug, banker in rms.items():
        brief = generate_brief(runner=runner, slug=slug, banker=banker)
        path = output_dir / f"{slug.replace('_','-')}.md"
        path.write_text(brief.markdown, encoding="utf-8")
        summary_rows.append(brief.summary_row)

    # Write index page
    index_md = generate_index(summary_rows)
    (output_dir.parent / "BANKER_BRIEFS.md").write_text(index_md, encoding="utf-8")

    print(f"Generated {len(summary_rows)} briefs in {output_dir}/")
    return 0
```

Per-banker brief generation queries:
1. Total customers: `SELECT COUNT() FROM Account WHERE OwnerId = :user_id AND External_ID__c LIKE 'HYDRATE-%'`
2. Persona mix: `SELECT FinServ__ClientCategory__c, COUNT(Id) FROM Account WHERE OwnerId = :user_id AND External_ID__c LIKE 'HYDRATE-%' GROUP BY FinServ__ClientCategory__c`
3. Open opps: `SELECT COUNT(Id), SUM(Amount) FROM Opportunity WHERE OwnerId = :user_id AND IsClosed = false AND External_ID__c LIKE 'HYDRATE-%'`
4. Open cases: `SELECT COUNT(Id) FROM Case WHERE OwnerId = :user_id AND IsClosed = false AND External_ID__c LIKE 'HYDRATE-%'`
5. Tasks this week: `SELECT COUNT(Id) FROM Task WHERE OwnerId = :user_id AND ActivityDate <= NEXT_N_DAYS:7 AND External_ID__c LIKE 'HYDRATE-%'`
6. Sample customers: `SELECT Id, Name, External_ID__c, FinServ__ClientCategory__c, FinServ__AnnualIncome__pc, FinServ__TotalInvestments__c FROM Account WHERE OwnerId = :user_id AND External_ID__c LIKE 'HYDRATE-%' ORDER BY FinServ__TotalInvestments__c DESC LIMIT 6`
