# Documentation index — DC AgentForce Output

**What this project does:** A Lightning card that **shows text your Flow already produced**—from Agentforce, Prompt Builder, or other automation. Users can **copy**, **expand**, or **print** the answer. Optional **thumbs up/down** if your Flow returns a **generation Id**.

The component does **not** replace Flow or Agent Builder; it **displays** the Flow output.
